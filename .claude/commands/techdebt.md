# /techdebt - 기술 부채 정리

프로젝트의 기술 부채를 찾아서 정리합니다.

## 수행할 작업

1. **미사용 Import 찾기**
   - 각 파일에서 import 되었지만 사용되지 않는 모듈 찾기
   - 자동으로 제거

2. **중복 코드 찾기**
   - 3곳 이상에서 반복되는 유사한 코드 패턴 식별
   - 공통 유틸리티/훅으로 추출 제안

3. **TODO/FIXME 주석 정리**
   - 프로젝트 전체의 TODO, FIXME, HACK 주석 목록화
   - 우선순위 제안

4. **타입 안전성 검사**
   - `any` 타입 사용 위치 찾기
   - 적절한 타입으로 교체 제안

5. **콘솔 로그 정리**
   - 디버깅용 console.log 찾기
   - 프로덕션에서 제거 필요한 것 식별

## 실행 방법

```
/techdebt
```

## 출력 형식

```markdown
## 기술 부채 리포트

### 🔴 높은 우선순위
- [파일:줄] 문제 설명

### 🟡 중간 우선순위
- [파일:줄] 문제 설명

### 🟢 낮은 우선순위
- [파일:줄] 문제 설명

### 💡 개선 제안
- 제안 내용
```
